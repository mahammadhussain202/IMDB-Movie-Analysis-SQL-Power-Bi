# 🎬 IMDb Movie Analysis using SQL & Power BI

## 📌 Project Overview

This project focuses on analyzing IMDb movie data using SQL and visualizing insights through an interactive Power BI dashboard. The analysis explores movie performance, director contributions, revenue trends, popularity, and audience ratings.

---

## 🛠 Tools & Technologies Used

* MySQL
* SQL
* Power BI
* GitHub

---

## 📂 Dataset Information

The dataset contains movie-related information including:

* Movie titles
* Revenue
* Popularity
* Ratings
* Release dates
* Director information

---

## 🔍 SQL Analysis Performed

The project includes:

* Basic SQL queries
* Aggregate functions
* Filtering and sorting
* JOIN operations
* Subqueries
* CASE statements
* Revenue and rating analysis

---

## 📊 Power BI Dashboard Features

### Dashboard Pages

1. Overview Dashboard
2. Director Insights Dashboard
3. Movie Performance Analysis

### Key Visualizations

* KPI Cards
* Revenue trends
* Top-rated movies
* Director performance analysis
* Scatter plots for popularity vs ratings
* Interactive slicers

---

## 🔑 Key Insights

* Avatar generated the highest revenue in the dataset.
* James Cameron is the most bankable director.
* Revenue and popularity show a positive relationship.
* Movie releases increased significantly after 2010.
